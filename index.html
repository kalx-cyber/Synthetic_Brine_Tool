Workflow

Use the tabs from left to right. The tool stores edits in this browser automatically.

1. Water tests: enter all lab tests. Blank cells are ignored; zero is treated as a measured zero.
2. Salt stock: tick the salts you have and set purity/assay if less than 100%.
3. Salt choices: pick alternatives such as NaBr vs KBr. The result updates automatically.

Water tests

Enter concentrations for as many tests as you have. Blank = not measured. Values like <0.05 are treated as half the detection limit.

Average and charge balance

Unit notes

The recipe calculations use mg/L as the aqueous species. Examples: nitrate as N is converted to nitrate ion, sulfate as S is converted to sulfate, and alkalinity as CaCO₃ is converted using equivalent weight.

Salt stock and purity

Tick salts that are available. Purity is used as a weighing correction. For example, 99% purity means mass to weigh = pure salt mass / 0.99.

Salt choices by target species

The dropdowns only appear for measured species with positive average concentration. If a salt is not in stock, choose another in-stock candidate.
